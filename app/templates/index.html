Using Yuzu Emulator (PC)

  1. Download and install Yuzu Emulator from the official website
  2. Download your game files (NSP, XCI, or NSZ format)
  3. Open Yuzu and go to File ‚Üí Install Files to NAND

  5. Once installed, the game will appear in your game library
  6. Double-click to launch and enjoy!
Note: You'll need Switch firmware files and prod.keys to run games. Check the Yuzu documentation for setup instructions.

Using a Modded Switch

  1. Ensure your Switch is properly modded with CFW (Custom Firmware) like Atmosphere
  2. Download your game files (NSP format recommended)
  3. Transfer the NSP files to your SD card's /switch/ folder or use a USB transfer tool
  4. Install games using TinWoo & NS USB Loader or similar tools
  5. Launch the game from your Switch home menu
Warning: Only use modded consoles offline or with appropriate precautions to avoid bans.
